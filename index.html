Github is a brilliant code hosting platform, that can be used to save our code projects and effectively tracking them. And also, to get a real world experience we can collaborate our software project with others. But, did you know that?. There is an another fantastic service is provided by Github.

In this blog we will learn about how to host our Static website on github for free by deploying them on github. It's very simple, easy and fast process, within few minutes our wesite will be live. As a learner or a developer this is a super good service right?!.

Note:

There is one limitation. Since it's a static service, we can deploy only static websites like portfolio, etc (like using HTML, CSS, Javascript). It doesn't support server side languages (like Node.js, Python, PHP, etc.).
